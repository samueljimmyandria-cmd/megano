📖 Tutoriel rapide

  1. Première utilisation — autorise l'accès à la caméra quand le navigateur le demande.
  2. Inscris-toi — appuie sur le bouton ➕, entre ton nom, capture ta main 3 fois sous différents angles.
  3. Ajoute un visage (optionnel) — améliore la sécurité, recommandé.
  4. Connecte-toi — depuis l'onglet d'identification, place ta main devant la caméra.
  5. Sécurité PIN — si le score est trop bas, le PIN de secours est demandé.

💡 Tous les calculs se font localement dans ton navigateur. Aucune image n'est envoyée à un serveur.

ℹ️ À propos

Hand + Face ID est une démo de reconnaissance biométrique 100% locale. Les modèles utilisés :

  • MediaPipe Hands (Google) — détection des 21 points-clés de la main
  • Face-API.js (face-api.js) — reconnaissance faciale via TensorFlow.js

Les données biométriques (vecteurs mathématiques) sont stockées dans le localStorage du navigateur, jamais transmises.

⚠️ Démo de recherche — ne pas utiliser comme seule sécurité pour des données sensibles.
